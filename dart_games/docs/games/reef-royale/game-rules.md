# Reef Royale — Game Rules

## Objective

Claim all 7 coral types on the reef and accumulate the most pearls to win. Each player is assigned a unique sea creature and competes to dominate the reef.

## Setup

- **Players:** 2-8
- **Target Numbers:** 20, 19, 18, 17, 16, 15, Bull (standard) or randomized (Random Reefs)
- **Mark Threshold:** 3 marks to claim (standard) or 2 marks (Easy Claim)

## Turn Structure

1. **Turn Start** — Active player highlighted, creature displayed
2. **Throw Darts** — Up to 3 darts per turn
3. **Process Hits** — Marks added, corals claimed, pearls scored
4. **Remove Darts** — Physical dartboard takeout or emulator button
5. **Next Player** — Advance to next player

## Dart Outcomes

### Marking (Unclaimed Target)
| Hit Type | Marks Added |
|----------|-------------|
| Single | 1 mark |
| Double | 2 marks |
| Triple | 3 marks |
| Inner Bull (50) | 2 marks |
| Outer Bull (25) | 1 mark |

When a target reaches the mark threshold (2 or 3), the coral **blooms** (is claimed by that player).

### Pearl Scoring (Claimed Target)
Once you've claimed a coral, hitting it again scores **pearls** equal to the target number × multiplier, as long as at least one opponent hasn't claimed it.

| Hit Type | Pearl Value |
|----------|-------------|
| Single 20 | 20 pearls |
| Double 20 | 40 pearls |
| Triple 20 | 60 pearls |
| Inner Bull | 50 pearls |
| Outer Bull | 25 pearls |

### Locking
When **all** players have claimed a target number, it **locks** — no more pearls can be scored on it.

### Miss / Non-Target
Hitting a number that isn't one of the 7 active targets scores nothing.

## Game Modes

### Standard Mode
- Score pearls for yourself
- Win by claiming all 7 corals with the most pearls (or by ranking when all targets lock)

### Cursed Tide Mode
- When you score pearls, they go to **all opponents** who haven't claimed that target
- Winner is determined by **fewest pearls** (pearls are bad!)
- Race to claim corals before opponents cash in on your hits

## Game Options

### Easy Claim
Only 2 marks needed to claim a coral instead of 3. Great for younger players.

### Neighbor Numbers
Adjacent numbers on the physical dartboard also count as hits. For example, if 20 is a target, hitting 1 or 5 (its neighbors) adds marks to 20. Shared neighbors (adjacent to two targets) add marks to both.

### Random Reefs
Shuffles the 6 number targets randomly each game (from 1-20). Bull is always the 7th target.

### Bonus Buffs
Random buffs activate at round boundaries (~33% chance):
- **Riptide Rush** — Marks doubled this round
- **Pearl Fever** — Pearl values doubled this round
- **Ink Cloud** — Opponent info hidden this round

### Show Hints
Displays the active target numbers in the player panel for reference.

### Speed Play
Sets a round limit. When reached, the game ends and the winner is determined by ranking.

## Win Conditions

1. **All Corals Claimed** — First player to claim all 7 corals wins (tiebreak by most pearls in standard, fewest in Cursed Tide)
2. **All Targets Locked** — When every target is locked, winner determined by ranking (most corals, then pearls)
3. **Speed Play Limit** — Round limit reached, winner by ranking

## Ranking Rules

Players are ranked by:
1. Most corals claimed (descending)
2. Most pearls (standard) or fewest pearls (Cursed Tide)

## Special Mechanics

- **Skip Turn** — Skip remaining darts for the current turn
- **Edit Score** — Adjust all 3 dart scores (corrects misreads)
- **Excess Marks** — Marks beyond the threshold on a claiming dart convert to pearl scoring
